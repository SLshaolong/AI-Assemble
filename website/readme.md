# Chat Application

这是一个简单的聊天应用程序,包含以下主要功能:

## 功能模块

### 聊天功能 (Chat)
- 支持用户之间的实时对话
- 保存聊天历史记录
- 聊天记录存储在 chat_history 目录下

### Token管理 (TokenManage) 
- 用户Token账户管理系统
- Token使用记录追踪
- 数据存储在 chat_utils/token_manage.json
